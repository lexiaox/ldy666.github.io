`memset(数组名, 0, sizeof(数组名));` 这个语句的作用是**将整个数组的所有字节都设置为0**。

## 函数原型
```c
void *memset(void *str, int c, size_t n);
```
- `str`: 要填充的内存块的指针
- `c`: 要设置的值（以int形式传递，但实际使用每个字节的低8位）
- `n`: 要设置的字节数

## 具体作用
```c
int arr[100];
memset(arr, 0, sizeof(arr));
```
这行代码将：
- `arr` 数组的 `100 * sizeof(int)` 个字节全部设置为0
- 相当于将数组的所有元素初始化为0

## 等价写法
```c
// 传统循环初始化
for(int i = 0; i < 100; i++) {
    arr[i] = 0;
}

// memset方式（更高效）
memset(arr, 0, sizeof(arr));
```

## 注意事项
1. **只能用于清零**：`memset(数组名, 0, sizeof(数组名))` 只适用于将数组清零
2. **非零值要小心**：如果设置非0值，结果可能不符合预期
   ```c
   int arr[3];
   memset(arr, 1, sizeof(arr)); // 不是设置元素为1！
   // 实际效果：每个int的4个字节都变成0x01
   // 即每个元素变成 0x01010101（16843009）
   ```

3. **适用于任何数据类型**
   ```c
   char str[100];
   double values[50];
   struct Student students[30];
   
   memset(str, 0, sizeof(str));     // 字符数组清零
   memset(values, 0, sizeof(values)); // 双精度数组清零
   memset(students, 0, sizeof(students)); // 结构体数组清零
   ```

## 使用场景
- 数组初始化
- 内存清零（安全考虑）
- 结构体清零

这是一种高效初始化数组的常用方法。
