# C语言示例

## 代码演示

```c
#include <stdio.h>
#include <string.h>

int main() {
    int arr[5];
    
    // 使用memset初始化数组
    memset(arr, -1, sizeof(arr));
    
    // 输出数组元素
    for(int i = 0; i < 5; i++) {
        printf("arr[%d] = %d\n", i, arr[i]);
    }
    
    return 0;
}
```

## 编译命令
使用 `gcc -o program program.c` 编译上述代码。
